# Uji T-Test pada Data Armada 
-Menggunakan Data dengan perbandingan yang signifikan
- Data yang digunakan merupakan Data dari GPS
- Fokus pada Distribusi BBM
- Data Lama merupakan data yang terdiri dari 2 Bulan, dengan uji T-Test tidak normal pada distribusi nya dengan kemungkinan berbagai alasan yakni: Rute, jarak, kondisi jalan
- Data Baru merupakan data yang hanya 4 hari. distribusi BBM normal namun faktor tersebut harus tetap di evaluasi karena datanya masih hanya beberapa trip juga rute yang dilalui tidak seberagam data lama
